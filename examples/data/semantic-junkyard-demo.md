# Semantic Junkyard Demo Corpus

Semantic Junkyard is an agent-native semantic layer that combines metadata graphs, semantic contracts, lineage, policy-aware retrieval, vector search, and knowledge graphs.

The platform supports full data ingestion, metadata-only registration, and external references via standard interfaces such as REST, OpenAPI, and MCP-style tool descriptors.

An autonomous agent can run discovery, search semantically, inspect graph paths, open evidence, and explain permission boundaries. It must stop when evidence is missing or when a task requires source mutation, restricted data, generated SQL execution, secrets, or external communication.
